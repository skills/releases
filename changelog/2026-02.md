# February 2026

Welcome to the February 2026 release notes for [GitHub Skills](https://learn.github.com/skills)! 🚀

This month focused on an overhaul of how images are managed. Previously, all images were included as [attachments](https://docs.github.com/en/get-started/writing-on-github/working-with-advanced-formatting/attaching-files) and referenced using absolute urls so they could be used in issues.

Now images are stored directly in the repository! This might be subtle change, but we think it will make creating exercises just a bit more natural! 🎉

### Highlights

- Image migration from external attachments to local assets
- Exercise Toolkit now uses GitHub App tokens instead of PATs
