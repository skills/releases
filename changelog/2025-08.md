# August 2025

Welcome to the August 2025 release notes for [GitHub Skills](https://learn.github.com/skills). 🎉

This month adds 2 new exercises: One to master your prompting in GitHub Actions, and another to help migrate from Azure DevOps. Additionally we have several minor updates to existing exercises, reorganized docs for developing (your own) new exercises, and a sneak peak at our plans to better connect with the community via an interactive exercise roadmap. 🧐

### Highlights

- 2 new exercises
- 6 updated exercises
- Exercise Manager moved to private preview
- Development preview of the Exercise Roadmap

## Exercises

### ✍️ New Exercises

> Learn how to migrate repositories from Azure DevOps to GitHub using the ado2gh CLI tool.

- Set up a mock Azure DevOps project.

- Migrate the Azure DevOps repository to GitHub.
- Integrate a GitHub repository with Azure DevOps Boards and Pipelines.

## 🧱 Platform

### Skills Exercise Toolkit (preview)

The [Skills Exercise Toolkit](https://github.com/skills/exercise-toolkit) is a vision to build a collection of commonly used components. Our goal is to simplify the underlying functional parts and provide space for more flexibility and personality in the exercises.

**Status:** `public preview` - working towards a `v1.0` release.

- Release of [`v0.7.1`](https://github.com/skills/exercise-toolkit/releases/tag/v0.7.1).
- Update dependencies for `actions/checkout` Action.
- Update dependencies for super linter.

### Skills Exercise Manager Codespace

The [Exercise Manager](https://github.com/skills/exercise-manager) Codespace is a pre-configured multi-workspace VS Code environment designed for easier authoring and updating of Skills exercises.
The vision is to provide consistent runtime environments, linting settings, test tools, recommendations from the [exercise toolkit](https://github.com/skills/exercise-toolkit), and super fast authorship via GitHub Copilot.

**Status:** `private preview` - Beta testing before public preview.

- Removal of experimental features.
- Refactoring of documentation for easier navigation within the codespace.

### Skills Exercise Roadmap

The **Exercise Roadmap** is an interactive Skills Tree style graph. It visually shares our plans for developing new exercises and the relationships between exercises. More importantly, it will provide an opportunity for the community to influence priority and design of upcoming exercises.

**Status:** `private` - Early experimental phase.

<img width="500" alt="exercise roadmap" src="https://github.com/user-attachments/assets/39adf55c-c820-4c8a-bfbf-6addb1dd2e1b" />

Learning Paths

- Colors are used to loosely show groups or learning paths.

Exercise Status

- Solid outline: Finished
- Dashed outline: Backlog (unscheduled)

  - Each exercise has a public Issue providing an outline of the exercise.
  - The community can like/dislike and add comments to influence priority and development.

- Dashed outline with calendar icon: Scheduled for development
- Solid color with spinner: In development

Simplistic search and filtering

- Top navigation provides text search across all fields.
- A filter menu allows manual exploring.
- Matching items stay full brightness. Non-matching items are dimmed.
