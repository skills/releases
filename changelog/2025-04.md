Welcome to the April 2025 release notes for GitHub Skills. This is our second release! Time is flying! ⏱️

This month, we have created 2 new exercises, updated 3, and made great quality of life improvements to the platform to make future exercise development easier. Maybe even fun! Read to the end! 🚀

## Exercises 

### ✍️ New Exercises

#### 🆕 [Integrate MCP with Copilot](https://github.com/skills/integrate-mcp-with-copilot/)

> Learn how to give GitHub Copilot more tools to expand the capabilities of your development workflow. All in less than an hour!

In this exercise, you will:

1. Integrate a GitHub MCP server with GitHub Copilot.
2. Delegate Copilot to research similar projects and open issues.
3. Ask Copilot to find an important issue and implement it from idea to pull request.
4. Add comments to a recently closed issue.

#### 🆕 [Introduction to Repository Management](https://github.com/skills/introduction-to-repository-management)
 
> Learn the basics of several GitHub features that can help support a collaborative, friendly, and healthy project.

In this exercise, you will:

1. Add a simple rulesets and configuration to restrict repository content.
2. Communicate procedures to help guide collaborators.
3. Assign responsibility of parts of the code to particular collaborators.
4. Learn the difference between collaboration in a personal repository and organization repository.
5. Establish ground rules to promote a health collaboration environment.
6. Establish a process for managing security updates.

### Skills Exercise Toolkit (preview)

The [Skills Exercise Toolkit](https://github.com/skills/exercise-toolkit) is a vision to build a collection of commonly used components.
Our hope is templatize the underlying functional parts and provide space for more flexibility and personality in the exercises.

**Status:** `public preview` - working towards a v1.0.0 release

#### [skills/action-text-variables@v2](https://github.com/skills/action-text-variables) 

> Replace mustache style variables (`{{ variable }}`) in text templates. Returns modified text as an output for use in other actions.

- Enables `YAML` style inputs and remvoes `ENV` style inputs.
- Adds several unit tests to for better reliability.
- Cleans up the CI workflow for easier versioning.

#### [skills/action-keyphrase-checker@v1](https://github.com/skills/action-keyphrase-checker)

> Search for keyphrases in text content or files. Validates if a specific phrase appears a minimum number of times with options for case sensitivity.

- Our first reusable grading action! 🎉
- A simple way to check the number of occurences of a word in a file. 
- Provides the found occurences for use in a feedback response.

#### General
- Provides markdown templates to support simplified start/finish exercise flow.
- Provides reusable workflows to support simplified start/finish exercise flow.
- Simplifies results table to `pass` and `message`. Removes "name" column.
- Deprecation of [skills/response-templates](https://github.com/skills/response-templates)

### Skills Exercise Development Codespace

The [Skills Exercise Development Codespace](https://github.com/skills/skills-manager) (currently known as "Skills Manager") is a preconfigured multi-workspace VS Code environment designed to easier authoring and updating of Skills exercises.
The vision is to provide consistent runtime environments, linting settings, test tools, and access to the Skills Toolkit to accelerate exercise development

**Status:** `private` - Working towards enough functionality and stability for private preview.

#### Exercise Development Guidelines

Many notes across several locations were brought together to provide our first set of detailed guidelines. The goal with these detailed guides is to enable future [Copilot-driven exercise development](https://github.com/skills/skills-manager#the-future).

These include:

- Planning Guide
- Formatting Guide
- Design Guide and Checklist
- Testing Guide
- Publishing Guide and Checklist
- Maintenance, Deprecation, and End-of-Lift Guide

#### Copilot-based Exercise Development

We made our firsta attempts at using Copilot to generate significant exercise content. This is aided by 2 new prompts support by the guidelines and the GitHub MCP server.

- Adds a prompt file to aid migration from readme-based to issue-based flow.
- Adds a Copilot prompt to generate a draft exercise step when provided an outline of the exercise via GitHub Issue URL.


- Simplified the workspace configuration to need less modifications for exercise development.
