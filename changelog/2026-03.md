## Exercises

### 🌟 New Exercise

#### [Work with Workflow Artifacts](https://github.com/skills/workflow-artifacts)  
> Learn how to upload, preview, download, and reuse workflow artifacts in GitHub Actions.

This new course teaches developers how to manage workflow artifacts effectively in GitHub Actions workflows. Key takeaways include uploading and reusing artifacts across jobs and gating production deployments with environment approvals. Complete the guided workflows to learn how to enhance your CI/CD practices in under an hour!
