# March 2026

Welcome to the March 2026 release notes for [GitHub Skills](https://learn.github.com/skills)! 🌱

This month brings a fresh exercise for mastering workflow artifacts, substantial refactors and improvements across core exercises, and platform upgrades for smoother automation, security, and developer experience. Get ready for big rewrites, modernization with Node.js 24, improved instructional clarity, and enhanced CI/CD reliability!

### Highlights

- **New:** Learn how to work with workflow artifacts in GitHub Actions
- **Major Refactor:** Issue-driven exercise rewrite for reusable workflows
- **ES Modules:** All JavaScript Actions code now uses modern ES module syntax
- **Devcontainer Upgrades:** Smaller, faster devcontainers—now on Ubuntu Jammy
- **Workflow & Dependency Updates:** Node.js 24, latest Actions versions, and security upgrades across key repos
- **Toolkit Enhancements:** New reusable action for workflow coordination and improved error handling
- **Exercise Documentation Improvements:** Instructional notes, UI references, and visual formatting clarified
