# July 2025

Welcome to the July 2025 release notes for [GitHub Skills](https://learn.github.com/skills). 🎉

This month builds on the momentum of the newly released [GitHub Learn](https://learn.github.com/). There are new exercises to further master Copilot, updated older exercises to refine your core skills, and slick updates to our platform for creating better exercises (and faster) in the future. Nice! 😎

### Highlights

- Migration to GitHub Learn
- 2 new Copilot exercises
- 8 updated exercises
- Enhanced templating for even better future exercises
- Automation to create exercises with Copilot

### 🚀 GitHub Learn Launch

[GitHub Learn](https://learn.github.com) officially launched on July 1st, providing a centralized hub for all GitHub learning resources. GitHub Skills is easily found via the top navigation bar and links to [learn.github.com/skills](https://learn.github.com/skills).

With this change, developers can discover hands-on exercises easier than ever to grow their GitHub expertise.

> [!IMPORTANT]
> In alignment with internal product requirements, the original [GitHub Skills](https://skills.github.com) site and related exercises have been reverted. All updated and new exercises are exclusively available via [GitHub Learn](https://learn.github.com/skills).

## Exercises

### ✍️ New Exercises

#### 🆕 [Modernize your Legacy Code with GitHub Copilot](https://github.com/skills/modernize-your-legacy-code-with-github-copilot)

> Modernize a school's legacy accounting system from Cobol to Node.js with the help of GitHub Copilot.

- Explain the current state of your legacy code.
- Create a diagram of the data flow in your code.
- Identify areas of your legacy code that can be improved.
- Use GitHub Copilot to generate modern code snippets.
- Replace the old code with the new snippets and test the changes.

> Customize GitHub Copilot's behavior with custom instructions, prompts, and chat modes for your specific development workflows.

- Set up repository-wide custom instructions to provide Copilot the essential project context
- Create targeted custom instructions for specific file types and directories
- Build reusable prompt templates for common tasks like homework assignments
- Configure custom chat modes for specialized workflows

## Platform

### Skills Exercise Toolkit (preview)

The [Skills Exercise Toolkit](https://github.com/skills/exercise-toolkit) is a vision to build a collection of commonly used components.
Our hope is templatize the underlying functional parts and provide space for more flexibility and personality in the exercises.

**Status:** `public preview` - working towards a `v1.0` release.

- Release of [`v0.7`](https://github.com/skills/exercise-toolkit/releases/tag/v0.7.0).
- Migrate from Mustache syntax to Nunjucks syntax. This enables more dynamic templates in the future.
- Simplify issue comment creation process with templates.

### Skills Exercise Manager Codespace

The [Exercise Manager](https://github.com/skills/exercise-manager) Codespace is a pre-configured multi-workspace VS Code environment designed for easier authoring and updating of Skills exercises.
The vision is to provide consistent runtime environments, linting settings, test tools, recommendations from the [exercise toolkit](https://github.com/skills/exercise-toolkit), and super fast authorship via GitHub Copilot.

**Status:** `private` - Final preparations for public preview.

- Rename from `skills-manager` to `exercise-manager`.
- Add Copilot custom instructions to guide exercise development.
- Add a Copilot prompt file to create a well-formed exercise outline following the GitHub Skills style.
- Add a Copilot prompt file to create a new well-formed exercise from an outline.
- Add a Copilot prompt to review an existing exercise against guidelines, best practices, and typical mistakes.
- Enable more environment variables when testing with Act.
- Switch to GitHub hosted MCP server.
- Add Context7 MCP server.
- Reduce start time by enabling Codespace pre-built container images.
- Simplify workspace structure to allow development flexibility.
