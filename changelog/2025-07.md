# July 2025

## Highlights

### 🚀 GitHub Learn Launch

GitHub Learn officially launched this month at [learn.github.com](https://learn.github.com), providing a centralized hub for all GitHub learning resources. GitHub Skills is now prominently featured as part of this new learning platform at [learn.github.com/skills](https://learn.github.com/skills), making it easier for developers to discover and access hands-on exercises to build their GitHub expertise.
