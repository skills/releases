# July 2025

## Highlights

### 🚀 GitHub Learn Launch

GitHub Learn officially launched this month at [learn.github.com](https://learn.github.com), providing a centralized hub for all GitHub learning resources. GitHub Skills is now prominently featured as part of this new learning platform at [learn.github.com/skills](https://learn.github.com/skills), making it easier for developers to discover and access hands-on exercises to build their GitHub expertise.

## 🧱 Platform

### Copilot Customizations for Exercise Development

The [Exercise Manager](https://github.com/skills/exercise-manager) now includes comprehensive Copilot customizations with custom instructions and prompt files specifically designed for GitHub Skills exercise development. These customizations help ensure consistent code style, proper exercise structure, and adherence to Skills best practices when using GitHub Copilot for exercise creation and maintenance.
