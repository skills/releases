# December 2025

Welcome to the December 2025 release notes for [GitHub Skills](https://learn.github.com/skills)! 🎁

This month features major platform upgrades, deeper documentation for enterprise users, and usability improvements across many top exercises. The focus: smoother developer experience, more robust automation, and expanded learning resources for everyone—from open source newcomers to managed enterprise organizations.

### Highlights

- 9 updated exercises with improved resources, grading, and step-by-step guidance
- Platform toolkit upgrades: faster, smarter CI/CD with modern dependencies and new features
- Expanded enterprise documentation for EMU organizations, including automated allowlist and permissions management
- Improved markdown templating, live preview, and developer container support for exercise authors
- Enhanced clarity and accessibility in exercise instructions and validation

## 🧱 Platform

### [Exercise Toolkit](https://github.com/skills/exercise-toolkit) [📝 changes](https://github.com/skills/exercise-toolkit/commits/main?since=2025-12-01&until=2025-12-31)

> Standardized toolkit for GitHub Skills exercises.

- Upgraded `peter-evans/create-pull-request` to v8—introduces Node 24 support and requires newer Actions Runner versions.
- Bumped `super-linter` to v8.3.2—gaining support for Rust 2024 edition, nbqa config files, and improved linting for GitHub Actions/Dependabot workflows.
- Numerous bug fixes: better handling for file paths, grouped logging, and dependency update organization.
- Extensive dependency and configuration refreshes for Java, Python, Docker, Rubocop, NPM, and more.
- Maintenance updates: improved documentation, upgraded issue templates, conflict awareness during upgrades.

**Result:** Fast, modern, and more reliable CI tooling for everyone building and using Skills exercises!
