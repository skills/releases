# December 2025

Welcome to the December 2025 release notes for [GitHub Skills](https://learn.github.com/skills)! 🎁

This month adds 1 new exercise to start versioning your source code with Git. Additionally, Skills now has a clear guide for use with [Enterprise Managed Users](https://docs.github.com/en/enterprise-cloud@latest/admin/concepts/identity-and-access-management/enterprise-managed-users)! 🚀

### Highlights

- 1 new exercise
- 8 updated exercises
- Documentation for deploying GitHub Skills in EMU organizations.

## Exercises

### ✍️ New Exercises

#### 🆕 [Introduction to Git](https://github.com/skills/introduction-to-git)

> Use Git version control to work on a game using command line (CLI) and VS Code

- Understand what version control is and why developers use it.
- Configure your Git identity.
- Create your first repository and make commits.
- View project history and compare file changes.
- Work with branches to experiment safely.
  Learn about Git collaboration concepts.
