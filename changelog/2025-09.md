# September 2025

Welcome to the September 2025 release notes for [GitHub Skills](https://learn.github.com/skills). 🎉

This month adds 2 new exercises: one on Copilot review techniques, and one on sharing knowledge with Copilot Spaces. Additionally, we've started refreshing Copilot content! Keep an eye out for updates! 🧑‍🚀

### Highlights

- 2 new exercises
- 10 updated exercises
- Dependency and platform updates for improved functionality and security

## Exercises

### ✍️ New Exercises

#### 🆕 [GitHub Copilot Code Review](https://github.com/skills/copilot-code-review)

> Leverage Copilot to scale feedback, promote code quality, and catch issues before they reach production.

- Use GitHub Copilot to review code directly in VS Code for immediate feedback.
- Request Copilot code reviews on pull requests.
- Customize Copilot's review considerations with repository-specific instructions.
- Configure automatic code reviews using repository rulesets.

---

#### 🆕 [Scale Institutional Knowledge Using Copilot Spaces](https://github.com/skills/scale-institutional-knowledge-using-copilot-spaces)

> Organize, share, and streamline knowledge to enhance project management and collaboration.

- Add a repository as a source to your Copilot Space.
- Add instructions to your Copilot Space.
- Create issues in the repository using Copilot Spaces.
- Explore and summarize project management process documentation.
- Update repository documentation based on insights and gaps discovered.
