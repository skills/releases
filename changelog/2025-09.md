# October 2023

Welcome to the October 2023 release notes for [GitHub Skills](https://learn.github.com/skills). 🎉  

This month's release includes one brand-new exercise, exciting updates to 13 existing exercises, and platform enhancements for improved usability, compatibility, and performance. Dive into the detailed rundown below to explore what's new! 🚀  

### Highlights  

- 1 new exercise: GitHub Copilot Code Review  
- 13 updated exercises with bug fixes, usability improvements, and new features  
- Platform updates for improved dependency support and streamlined workflows  

---

## Exercises  

### ✍️ New Exercises  

#### 🆕 [GitHub Copilot Code Review](https://github.com/skills/copilot-code-review)  

> Leverage Copilot to scale feedback, promote code quality, and catch issues before they reach production.  

- Review code directly with Copilot in VS Code for real-time feedback.  
- Configure Copilot for pull request reviews with custom criteria.  
- Set up automatic code reviews using repository rulesets.  
- Enhance a real-world project, the Mergington High School website!  

This exercise helps developers familiar with GitHub Copilot streamline their code review workflows while exploring automated review capabilities with a practical focus.
