# April 2026

Welcome to the April 2026 release notes for [GitHub Skills](https://learn.github.com/skills)! 🎉

This month’s update is packed with feature-rich updates and enhancements across various GitHub Skills exercises and platform tools. From modernized instructional content to the addition of GitHub Copilot Chat in over a dozen repositories, these changes aim to improve the learning experience and developer productivity.

### Highlights

- **Modern and enhanced learning experiences** with updated content and expanded features across multiple exercises.
- **GitHub Copilot Chat extension** was integrated into 15 exercises, enabling richer conversational AI interactions for learners.
- **Platform enhancements** include new "Repo Agent Skills" for easier exercise creation and updates to core tools for improved workflows and compatibility.
