# November 2025

Welcome to the November 2025 release notes for [GitHub Skills](https://learn.github.com/skills). 🚀

This month, we're excited to introduce a brand-new exercise that lets you craft your own GitHub JavaScript Actions from scratch—making automation more accessible than ever! We also focused heavily on keeping our existing exercises current, improving documentation, updating visuals, modernizing agent workflows, and ensuring users are pointed to the latest features and resources. The Skills platform itself also received important reliability and security updates, helping everything run smoother behind the scenes.

### Highlights

- 1 new exercise: Write Your Own JavaScript Action
- 5 updated exercises: enhanced content, better visuals, and modernization for today's GitHub Copilot and Spark features
- Copilot Extensions deprecated in favor of Model Context Protocol—be sure to check the new guidance

## Exercises

### ✍️ New Exercises

#### 🆕 [Write JavaScript Actions](https://github.com/skills-dev/write-javascript-actions)

> Write your own GitHub JavaScript Action and automate customized tasks unique to your workflow.

- Learn to initialize a JavaScript project for actions, write source code for a simple action (like fetching dad jokes!), and bundle it for distribution.
- Define your action with `action.yml` and put your custom automation straight into a workflow.
- Test your action via workflow triggers—for example, using issue comments.
- Designed for beginners: you only need basic GitHub Actions knowledge. In less than an hour, you’ll be able to author, package, and use custom JavaScript Actions to supercharge your CI/CD pipelines.

## 🧱 Platform

### Skills Exercise Toolkit

The [Skills Exercise Toolkit](https://github.com/skills/exercise-toolkit) powers and standardizes our learning experiences. This month brings upgrades for security, reliability, and clearer documentation.

- Upgraded [super-linter](https://github.com/super-linter/super-linter) from v8.2.0 to v8.2.1:
  - Improves error handling, bug fixes for CI events, and skips linter errors on incorrect files.
  - Updated dependencies for Docker, npm, Python, and Ruby—plus new documentation for variables and tool usage.
- Upgraded [actions/checkout](https://github.com/actions/checkout) from v5 to v6:
  - Credentials are now stored more securely, requiring the latest Actions Runner for container compatibility.
  - Documentation updates, Node.js 24 support, and general cleanup.

**Impact:** Stability, security, and future-proofing for all Skills exercises—no breaking changes for existing workflows.
