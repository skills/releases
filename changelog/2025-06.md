
## Exercises

### ✍️ New Exercise

No new exercises were published this month. However, 12 new exercises from the [Berkley Business Society](https://www.berkeleybusinesssociety.com/) have been made on the topic of project management. They are in review to be officially migrated. They can can be seen in the [Skills Dev](https://github.com/skills-dev).

Tentative Titles:

- Intro to GitHub Releases
- Creating Semantic Version Tags
- Release Notes and Changelogs
- Actions for Releases and Deployments
- Intro to GitHub Issues
- Collaborating with Issues
- Customizing and Managing Issues
- Intro to GitHub Discussions
- Pinning Threads and Responses
- Intro to GitHub Projects
- Customizing Project Boards
- Automation for Project Boards
