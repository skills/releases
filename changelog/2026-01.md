## Exercises

### ✍️ New Exercises

#### 🆕 [Create applications with the Copilot CLI](https://github.com/skills/create-applications-with-the-copilot-cli)

> Use GitHub Copilot CLI to manage issues and build a calculator app.

- Learn to install the standalone Copilot CLI and use an issue template
- Use Copilot CLI to create an issue for a Node.js calculator app from an existing template
- Practice iterative development by working together with Copilot on the command line

#### 🆕 [Publish Docker Images](https://github.com/skills/publish-docker-images)

> Build and publish Docker images using GitHub Actions.

- Set up a development environment and create a basic build and publish workflow
- Enhance the workflow with specialized Docker actions for improved performance
- Implement dynamic tagging for automated versioning based on Git context
