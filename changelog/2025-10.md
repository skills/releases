# October 2025

Welcome to the October 2025 release notes for [GitHub Skills](https://learn.github.com/skills). 🎉

This month, we've introduced an exciting new exercise that will help you transform your ideas into web applications with zero coding experience—powered by GitHub Spark! Along with this, we've made significant updates to several existing exercises to improve functionality, user experience, and accessibility. Plus, there are powerful updates to our platform tools, including new features and dependency upgrades, to enhance the Skills experience overall. 🚀

### Highlights

- 1 new exercise
- 6 updated exercises
- Support for "Beast Mode" in exercise manager

---

## Exercises

### ✍️ New Exercises

#### 🆕 [Idea to App with GitHub Spark](https://github.com/skills/idea-to-app-with-spark) 

> Turn your thoughts into real web apps by simply describing them, no coding required!

- Discover how GitHub Spark makes app development accessible to everyone, not just programmers.
- Select an example scenario and create your first web application.
- Refine and customize your app through conversation.
- Share your application with your workplace or community.
- Explore advanced features and underlying code.
